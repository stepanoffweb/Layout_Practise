Адаптивная HTML вёрстка из Adobe XD на реальном примере. Проект S&Mitler. Джедай вёрстки #7.

Важные примечания к выпуску:
- Готовый результат в папке "ready-html" поставляется "как есть", не стоит обновлять пакеты и вносить изменения в какие-либо настройки демонстрационного проекта. Пакеты устанавливаются обычным способом.
- Вёрcтка макета осуществляется с использованием Gulp 3. Если вы желаете обновить Gulpfile.js для работы с 4 версией, ознакомьтесь с этим видео: https://youtu.be/kesyrCZE1bA или можете просто использовать последнюю версию стартера для вёрстки. Функционал стартера может отличаться от представленного на видео, поэтому рекомендую перед вёрсткой ознакомитсья с уроком по последнему стартеру. Последнее видео в плейлисте "Стартер": https://www.youtube.com/playlist?list=PLyf8LgkO_8q9Uc8EagrBQOghHpDfj0LhI
- Если вы случайно обновили Gulp до версии 4 вместе с остальными пакетами и в процессе запуска gulp у вас возникают ошибки, откройте файл package.json и замените версию Gulp на 3.9.1 ("gulp": "^3.9.1",) - строка 14 в демонстрационном проектк "ready-html". После этого можно удалить node_modules и установить пакеты заново.
- Наименование папки вашего аккаунта Windows должно быть написано латиницей, например: "C:\Users\alex", а не кириллицей "C:\Users\алекс". Это необходимо для корректной работы Gulp, Prepros и Adobe XD.
- Adobe XD разрабатывался для платформы магазина Windows, поэтому если вы пользователь Windows, необходима вервия ОС Windows 10.
